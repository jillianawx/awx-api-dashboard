Airwallex Quickstart

Purpose of this tool is to test out the different API calls. Pertaining to Payments, Fund Split and Payouts.

Here are some tips to help you navigate this project:

1. Before getting started:

Please provide the following details in server/helper.js: Client ID, API Key, Scale Model (1.x/2.x), if applicable
Note: For payment pages and issues, please refer to original documentation on the airwallex-payment-demo github

2. Tools in this demo

Core
- Wallet, Deposits, Reports
- Global Accounts
- Linked Accounts

Scale
- Scale (Connected Accounts and Platform Reports)
- KYC/KYB

Payments
- Payment Links, Intents, Attempts, Refunds
- Payment - Embedded and Drop In
- Funds Splits and Reversals

Payouts
- Payouts and Beneficiaries

3. Additional notes

INTERNAL LOCAL USE ONLY

Please move all API calls to a server for deployment. Do not push API keys into repositories.

This is a hacky demo for internal use and testing only. Any code is not meant for production use and is not thoroughly tested.
